# UC: 問題の詳細を見る

## 基本コース

- ユーザーは任意のページから任意の問題について詳細ボタンをクリックする
- システムは指定された問題について，問題 ID をもとに問題リストに問題情報を問い合わせる
- 問題リストは問題情報と解答リストへのパスをシステムに提供する
- システムは取得した問題情報をもとに解答をいくつか問い合わせる
- 解答リストは解答をリアクションの多い順に提供する
- システムは取得した情報を基にページを構成する

## 代替コース

### 問題 ID に対応する問題が存在しなかった場合

- 問題リストはシステムに未定義例外を投げる
- システムは「問題が見つかりませんでした」と表示する

### 解答リストに解答が存在しなかった場合

- 解答リストはシステムに問題が存在しない旨を伝える
- システムは「解答がまだ存在しません」と表示する
- システムは取得した情報を基にページを構成する
